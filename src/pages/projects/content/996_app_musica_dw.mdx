Se ha desarrollado un Datawarehouse con el modelo copo de nieve (el análisis de las tablas de dimensión se realizó con cubos OLAP),
usando la base de datos Oracle y su lenguaje PL/SQL para el desarrollo de los objetos a utilizar en el proceso ETL.
Los datos de la app se econtraban almacenados en un archivo CSV y se ha gestionado el proceso ETL creando un orquestador con el lenguaje
Java. Y se ha usado Tableau como herramienta DSS para la visualización de datos y toma de decisiones.
