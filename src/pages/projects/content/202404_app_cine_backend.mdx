Este proyecto consiste en el desarrollo de una <strong>API-REST</strong>, uso de <strong>microservicios</strong> para la separación en distintos servidores
de los módulos del negocio, y conexión con el motor de base de datos relacional MySQL se ha creado la lógica en parte del
servidor para la aplicación de un cine. Se ha hecho uso de Swagger para la documentación de los endpoints que contienen las APIs,
y Git/GitHub para el control de las versiones del proyecto.
