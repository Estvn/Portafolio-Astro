Se ha desarrollado una aplicación web completa para la gestión de citas médicas a los doctores de un hospital oncológico. 
Se ha utilizando Javascript y Bootstrap para el desarrollo en el FrontEnd, tecnología AJAX para el consumo de endpoints 
y librerías para la creación de archivos CSV y PDF con información almacenada en la base de datos de la aplicación.
Y se ha desarrollado una API-REST con PHP vanilla con el patrón de diseño <strong>MVC</strong> y conexión con la base de datos MySQL 
para el manejo de lógica en la parte del servidor.
