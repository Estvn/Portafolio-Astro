Se ha desarrollado una aplicación web completa para gestionar la contabilidad de una cadena de iglesias a nivel nacional.
Se ha utilizado Javascript, Bootstrap, jQuery y varias librerías generadoras de archivos CSV y PDF para el desarrollo 
del FrontEnd de la aplicación. Y la creación de una API-REST con PHP, 
el patrón de diseño MVC y conexión con el gestor de base de datos MySQL para el manejo de la lógica en el servidor.
