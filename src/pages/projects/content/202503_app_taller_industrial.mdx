Se ha desarrollado una aplicación web completa para la gestión del inventario de un taller Industrial. La aplicación contiene los diferentes
roles (administrador, técnico, cliente) y sus respectivas autorizaciones y autenticaciones para garantizar la seguridad en la visualización 
de datos entre los diferentes roles. Se ha usado Javascript, Bootstrap y jQuery en el FrontEnd, se ha creado una API-REST con PHP y conexión 
con el motor de base de datos MySQL para la lógica en la parte del servidor.
